✨ 核心特性

🔗

多渠道聚合

Exa / Tavily / TinyFish，YAML 配置即可扩展新渠道，无需改代码。

⚡

优先级路由

按 priority 顺序依次尝试，搜到结果即停，节省上游额度。

🔄

自动故障切换

渠道失败自动切换下一个，Key 级 round-robin 负载均衡。

🛡️

双重限制

频率限制（滑动窗口）+ 配额限制（累计计数），防止滥用。

👥

用户体系

admin / normal 角色，JWT 认证，API Key 鉴权，按次计费。

🖥️

Web 管理面板

可视化配置渠道、Key、用户、监控调用日志，支持 CSV 导出。

💡 一次请求，多源聚合

统一的 RESTful API，返回标准化搜索结果

# 请求搜索 curl -X POST http://localhost:8830/api/search \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{"query": "AI agent framework", "max_results": 10}' # 响应结果（自动聚合去重） { "results": [ { "title": "Building AI Agents with LangChain", "url": "https://example.com/ai-agents", "snippet": "A comprehensive guide to...", "source": "exa", "score": 0.95 } ], "total": 10, "latency_ms": 1200 }

🚀 快速开始

1

克隆项目

git clone https://github.com/bluestar0123/search-aggregator.git cd search-aggregator
2

安装依赖

python -m venv .venv source .venv/bin/activate pip install -r requirements.txt
3

配置环境

cp .env.example .env # 编辑 .env 填入 JWT_SECRET # 编辑 config/channels/ 配置搜索渠道
4

启动服务

python main.py # 访问 http://localhost:8830
